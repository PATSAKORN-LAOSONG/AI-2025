# AI-2025
<img width="1005" height="689" alt="image" src="https://github.com/user-attachments/assets/04ceecf7-a97c-405b-be79-77b593b9a96b" />


ระบบนี้ถูกพัฒนาขึ้นเพื่อช่วยแนะนำการเรียนรู้ในแต่ละทักษะทางคณิตศาสตร์ของผู้เรียน โดยประกอบด้วยทักษะพื้นฐาน 4 ด้าน ได้แก่ การบวก การลบ การคูณ และการหาร ระบบจะทำการวิเคราะห์ จุดอ่อนของผู้เรียนในแต่ละทักษะ จาก คะแนนจริงที่ได้จากการทำแบบทดสอบ จากนั้นนำข้อมูลดังกล่าวเข้าสู่กระบวนการวิเคราะห์ด้วย ปัญญาประดิษฐ์ (AI) โดยใช้ โมเดล Random Forest เพื่อช่วยจำแนกและทำนายว่าผู้เรียนควรพัฒนาทักษะด้านใดเป็นพิเศษ และมีการวิเคราะห์ด้วยคะแนนจริงที่สอบในการจำแนกว่าผู้เรียนควรพัฒนาทักษะใด

## ฟีเจอร์หลัก (Features)
1) ระบบจำแนกและทำนายจุดอ่อนทางด้านคณิตศาสตร์ของผู้เรียน
   - มีการจำแนกโดยปัญญาประดิษฐ์ (AI) โมเดล Random Forest
   - มีการนำคะแนนจริงที่สอบมาใช้ในการจำแนกว่าผู้เรียนควรพัฒนาทักษะใดเพิ่ม
2) ระบบการทำข้อสอบเพื่อหาจุดอ่อนทางด้านคณิตศาสตร์ของผู้เรียน

## 💡 สะท้อนคิด
จากการทำโครงงานครั้งนี้ ข้าพเจ้าได้เรียนรู้เกี่ยวกับการทำงานของโมเดล AI และการเลือกใช้โมเดลให้เหมาะสมกับหัวข้อโครงงาน รวมถึงการนำ AI มาช่วยแก้ปัญหา การใช้ AI เป็นเครื่องมือช่วยในการทำงาน การค้นหาข้อมูลที่มีคุณภาพและสอดคล้องกับโครงงาน
ในส่วนที่ควรพัฒนาเพิ่มเติม คือการศึกษาการสร้างระบบ AI ให้มีประสิทธิภาพมากยิ่งขึ้น การเรียนรู้โมเดลที่หลากหลายมากขึ้น การทำงานให้เป็นไปตามแผนที่วางไว้ และการเพิ่มความรอบคอบในการทำงาน รวมถึงการค้นคว้าข้อมูลให้กว้างและลึกมากยิ่งขึ้น
